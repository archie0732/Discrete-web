# Discrete-web

#### download and use

```bash
git clone https://github.com/archie0732/Discrete-web.git
```
then,
```bash
npm run serve
```

**make sure you dir path and already installed npm** 

## about the website path 

the web 

```
./src/App.vue
```

and css

```
./src/styles.css
```

and script
```
./src/index.js
```
